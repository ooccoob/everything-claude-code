# 重构清理

透过测试验证安全地识别和移除无用程序代码：

1. 执行无用程序代码分析工具：
   - knip：找出未使用的 exports 和档案
   - depcheck：找出未使用的相依性
   - ts-prune：找出未使用的 TypeScript exports

2. 在 .reports/dead-code-analysis.md 产生完整報告

3. 依嚴重性分类发现：
   - 安全：测试档案、未使用的工具
   - 注意：API 路由、元件
   - 危险：设置档、主要进入点

4. 只提议安全的删除

5. 每次删除前：
   - 执行完整测试套件
   - 验证测试通过
   - 套用变更
   - 重新执行测试
   - 如果测试失敗则回滚

6. 显示已清理项目的摘要

在执行测试前绝不删除程序代码！
